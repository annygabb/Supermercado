# Supermercado
Desenvolvi um projeto pessoal para resolver uma dor real do meu cotidiano: o controle financeiro e o histórico de preços em compras de supermercado.
Decidi criar uma ferramenta própria para centralizar e analisar esses dados.

O que o projeto já faz:

Filtros Inteligentes: Busca por produtos específicos ou por estabelecimentos.

Análise de Custo-Benefício: O sistema identifica em qual supermercado o item está mais barato ou mais caro.

Visão Geral de Gastos: Cálculo do valor total acumulado por mercado ou o montante global de todas as compras.
